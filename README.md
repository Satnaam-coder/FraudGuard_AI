# 🚀 FraudGuard AI System

An AI-powered Fraud Detection Web Application built using Machine Learning and Flask.

---

## 📌 Project Overview

FraudGuard AI is a real-time fraud detection system that analyzes financial transactions and predicts the risk level using a trained machine learning model.

---

## 🧠 Features

- 🤖 Machine Learning Model (Random Forest)
- ⚖️ Handles imbalanced data using SMOTE
- 🔍 Fraud Probability Prediction
- 🚨 Risk Classification (High / Medium / Low)
- ⚙️ Decision Engine (Block / OTP / Allow)
- 🔄 Automated Transaction Simulation
- 📊 Live Transaction Monitoring Dashboard
- 🎨 Interactive Web UI using Flask

---

## 🛠️ Tech Stack

- Python
- Flask
- Scikit-learn
- Pandas
- HTML, CSS

📊 System Workflow 

* Generate transaction data
* Predict fraud probability using ML model
* Classify risk level
* Apply decision rules
* Display results in UI
